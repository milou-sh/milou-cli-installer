# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Unreleased (main) | ✅ |
| Tag releases (latest two) | ✅ |

Older tags may receive best-effort support only.

## Reporting a Vulnerability

1. Email security@milou.sh with a detailed description, affected versions, and any proof-of-concept material.
2. Encrypt mail using the team PGP key published at https://docs.milou.sh/security whenever possible.
3. Allow up to 48 hours for acknowledgement. Please do not open public issues for vulnerabilities until we have coordinated a fix.

Thank you for helping us keep Milou deployments secure.
